# Projeto One Piece

Este projeto é uma demonstração interativa dos personagens do anime One Piece. Ele permite aos usuários selecionar personagens e obter informações sobre eles.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
